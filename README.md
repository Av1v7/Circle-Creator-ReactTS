# Circle-Creator-ReactTS
Hello there, This is my circle creator project built in ReactJS, Hope you will enjoy.

### Steps to download:
  1. Download files as a rar files,
  2. Extract files.
  3. Open cmd\vscode terminal
  4. write "npm i"

  When the download have been completed
  
  5. write npm start
